# 💌 Capsule Mail

> Write letters to your future self or someone you love. Delivered by email on the date you choose.

## 🚀 Features
- Schedule letters for future delivery
- Send to yourself or someone else
- Attach images
- Letters are locked until delivery date
- Email notification on delivery

## 🛠 Tech Stack
- **Backend:** Django, Django REST Framework
- **Task Queue:** Celery + Redis
- **Frontend:** React
- **Email:** SendGrid
- **Storage:** Cloudinary
